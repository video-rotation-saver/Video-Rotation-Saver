# Video Rotation Saver Branding Assets

These SVG masters are based on the approved Concept 2 direction from the branding board, but remove the repeated `90` angle label from the icon/logo artwork.

Use these cleaned assets for packaging:

- `app-icon-clean.svg` - app, tray, taskbar, Start Menu, and installer icon source.
- `logo-horizontal-clean.svg` - main large-use logo.
- `splash-clean.svg` - optional startup splash source.
- `installer-banner-clean.svg` - installer header/banner source.

The original branding board remains useful as visual direction, but these cleaned files should be treated as the package-ready source assets.
